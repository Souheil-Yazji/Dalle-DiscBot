# Dalle-DisBot
A Discord bot that generates images through the OpenAI Images API.

## Setup
1. Install dependencies with `npm install`.
2. Create a `.env` file with:
   - `DISCORD_BOT_TOKEN`
   - `OPENAI_API_KEY`
3. Start the bot with `npm start`.

## Usage
- Send a message starting with `!dalle` followed by a prompt.
- Or use the `/dalle` slash command with a prompt.
